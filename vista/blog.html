Proyecto Autene.

  • Autor : AntonioHH
  • 03 Abr, 2016
  • Aplicaciones Web

Vídeo demostración del funcionamiento de la aplicación Autene.

La aplicación consiste en una solución de cálculo del autoconsumo de energía eléctrica mediante instalaciones fotovoltaicas en viviendas aisladas de la red. Autene se desarrolló gracias a PHP, lenguaje de código del lado del servidor. El lenguaje Javascript para el lado del cliente, complementando a PHP, y para establecer comunicación entre ellos, Xajax, el Ajax de PHP. El aspecto visual es de Materialize, framework en HTML5, CSS3 y Javascript, apoyandose en JQuery, Alertify en Javascript y PHPExcel para las funciones de exportación de los resultados de calculos almacenados. Despliegue Autene en Arquitectura LAMP.

Autene vive en Debian, corre con Apache y aprende con MySQL.

La aplicación Autene esta accesible en internet: Autene

Arduino Mega 2560 con módulo RF APC220 y App Java Led4Serial.

  • Autor : AntonioHH
  • 17 Mar, 2016
  • Arduino

Comunicación entre dos dispositivos Arduino Mega con módulos APC220.

En este video pueden ver una muestra de un nuevo proyecto que estoy desarrollando, combinando la programación en el lenguaje Java SE y la programación de los Dispositivos Arduino. Utilizando las antenas de RF es posible controlar los elementos conectados a los dispositivos arduino desde un ordenador en la oficina utilizando una aplicacion Java. Un arduino de control que activa los pines 2, 3, 4, 5 a los cuales están conectados cuatro diodos led, recibiendo del otro arduino, que actúa como TTL, conectado al ordenador, las ordenes específicas para encender o apagar los leds.

En el ordenador, la aplicación java Led4Serial realiza el envío de las ordenes que identificará el arduino de control segun su programación.

La programación del arduino de control y la aplicación java Led4Serial: https://github.com/antoniohh
