Debian VirtualBox - Instalación Sistema, Extension Pack y Guest Additions.

En el siguiente vídeo, instalamos el sistema operativo libre Debian, en su versión 8.4, como huésped, en una máquina virtual del software de virtualización de Oracle VirtualBox, versión 5.0.18. El sistema anfitrión es un Intel Xeon con 8Gb Ram y sistema operativo Windows 10 Pro x64. Descripción del proceso:

  • 1.- Instalación de VirtualBox 5.0.18 for Windows hosts.
  • 2.- Instalación de VirtualBox 5.0.18 Oracle VM VirtualBox Extension Pack.
  • 3.- Configuración de la máquina virtual.
  • 4.- Instalación del sistema operativo libre Debian 8: 8.4 released, 2 de abril de 2016.
  • 5.- Primer inicio del sistema operativo en la máquina virtual.
  • 6.- Actualizamos el sistema e instalamos gcc, make, dkms con los headers del kernel y copiamos el contenido de las Guest Additions en una carpeta en el home del usuario creado.
  • 7.- Damos los permisos de ejecución necesarios e instalamos las Guest Additions, comprobando que todos los procesos se realizan correctamente.
  • 8.- Finalmente comprobamos con el comando htop, el cual instalamos previamente, y con el monitor de recursos de mate el consumo de recursos de la máquina virtual.

Phopi (Php Orange Pi).

  • Autor : AntonioHH
  • 10 Abr, 2016
  • Aplicaciones Web - Orange Pi

App Web para el control de puertos Gpio, Phopi (Php Orange Pi).

El siguiente vídeo describe el proceso de configuración del servidor LAMP creado en el dispositivo Orange Pi PC. Configuración de los parámetros de la red, instalación de un cliente git, instalación de make y de gcc, activación de los puertos Gpio con las librerías compiladas WiringOP. instalación del servidor FTP ProFTPd y upload de la aplicación web Phopi al dispositivo a través del protocolo FTP con NetBeans y Filezilla.

La aplicación web Phopi (PHP Orange Pi) es una aplicación escrita en PHP, apoyandose en Xajax 0.5, Bootstrap 3.3.6 y JQuery 2.2.3, corriendo sobre un dispositivo Orange Pi PC, donde tenemos desplegado un servidor LAMP (Ubuntu Mate 15.04.1, Apache 2.4.10, MySQL 5.6.28 y PHP 5.6.4). Mediante la aplicación controlamos cuatro diodos led conectados a los puertos GPIO 22, 23, 24, 25 de la Opi.

Para ampliar información o consultar el código de la aplicación: https://github.com/antoniohh/Phopi.git

Instalación Ubuntu Mate 15.04 Orange Pi PC.

  • Autor : AntonioHH

Instalación Ubuntu Mate 15.04 Orange Pi PC. Despliegue servidor LAMP.

El siguiente video describe el proceso de instalación de la distribución GNU-Linux Ubuntu Mate 15.04 en el dispositivo Orange Pi PC. Se detalla la descarga de la imagen, volcado de la misma en la tarjeta de memoria, redimensionamiento de la partición y puesta en marcha del dispositivo Orange Pi.

Una vez iniciado el dispositivo, realizamos una conexión SSH con la aplicación Putty mediante su IP, y procedemos a la actualización del sistema y la instalación de la plataforma LAMP. Por último realizamos una conexión de escritorio remoto RDP con Remmina. Accedemos al escritorio de la Orange Pi y abrimos el monitor de sistema para ver los recursos disponibles.

Realizada la instalación de la plataforma, comprobamos que los servicios apache2 y mysql están activos y probamos con el navegador web el acceso a los mismos.

Proyecto Autene.

  • Autor : AntonioHH
  • 03 Abr, 2016
  • Aplicaciones Web

Vídeo demostración del funcionamiento de la aplicación Autene.

La aplicación consiste en una solución de cálculo del autoconsumo de energía eléctrica mediante instalaciones fotovoltaicas en viviendas aisladas de la red. Autene se desarrolló gracias a PHP, lenguaje de código del lado del servidor. El lenguaje Javascript para el lado del cliente, complementando a PHP, y para establecer comunicación entre ellos, Xajax, el Ajax de PHP. El aspecto visual es de Materialize, framework en HTML5, CSS3 y Javascript, apoyandose en JQuery, Alertify en Javascript y PHPExcel para las funciones de exportación de los resultados de calculos almacenados.

Autene vive en Debian, corre con Apache y aprende con MySQL.

La aplicación Autene está accesible en internet: Autene

Arduino Mega 2560 con módulo RF APC220 y App Java Led4Serial.

  • Autor : AntonioHH
  • 17 Mar, 2016
  • Arduino

Comunicación entre dos dispositivos Arduino Mega con módulos APC220.

En este video pueden ver una muestra de un nuevo proyecto que estoy desarrollando, combinando la programación en el lenguaje Java SE y la programación de los Dispositivos Arduino. Utilizando las antenas de RF es posible controlar los elementos conectados a los dispositivos arduino desde un ordenador en la oficina utilizando una aplicacion Java. Un arduino de control que activa los pines 2, 3, 4, 5 a los cuales están conectados cuatro diodos led, recibiendo del otro arduino, que actúa como TTL, conectado al ordenador, las ordenes específicas para encender o apagar los leds.

En el ordenador, la aplicación java Led4Serial realiza el envío de las ordenes que identificará el arduino de control segun su programación.

La programación del arduino de control y la aplicación java Led4Serial: https://github.com/antoniohh
